# این یک پروژه ساده TodoList می باشد
## تکنولوژی های مورد استفاده در این پروژه شامل موارد زیر می باشد:
- Html
- CSS
- JavaScript
- React
    - Redux
    - HookForm/Yup
    - axios
    - react-router
- Framer-Motion

## قابلیت های پروژه:
- Responsive/Dynamic Design
- Animated Sidebar
- Animated LandingPage
- TodoList Page
    - Edit
    - Delete
    - AddNewTask
- Login/Register
    - Validation

## توضیح پروژه
این یک پروژه ساده TodoList هست و هدف از این پروژه تمرین و استفاده از تکنولوژی های مختلف در آن هست . این پروژه با استفاده از موارد ذکر شده در بالا ساخته شده است و امکانات اولیه مانند ویرایش و حدف و اضافه کردن در لیست کارها را به کاربر میدهد وهمچنین دارای ظاهری پویا و رسپانسیو می باشد.
